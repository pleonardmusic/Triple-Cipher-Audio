DESCRIPTION

This web application takes an audio file as input. After the file is processed using a custom algorithm, the program generates three specially coded white noise files, which when played together regenerate the original sound file. It has potential for real-time audio encryption, but also has applications for music and art.

Triple Cipher Audio


Load a .wav or .MP3 file

After loading an audio file YOU MUST hit Process Audio below to initialize the function!!
